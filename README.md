# CivicPlus Internal Toolkit

An extension for Google Chrome for internal use by CivicPlus employees.

## Getting Started

You can add the extension to Chrome via the [Chrome Web Store](https://chrome.google.com/webstore/detail/civicplus-internal-toolki/pkckllogilffhjmgljfmllfmbgacjdla).

## Versions

See the [releases on GitHub](https://github.com/pkmnct/cp-toolkit/releases) for version history.
